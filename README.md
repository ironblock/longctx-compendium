# Long Context Falloff Compendium

An interactive visualization and data page examining long-context
inference performance across GPU platforms and model architectures.

**⚠️ THIS IS A VIBE-CODING EXPERIMENT.**

The entire page — every number, chart, caption, and verdict — was
generated by LLMs: Claude, Big Pickle, Gemma 4, and Qwen 3.6. I
(the human) asked questions, pointed at contradictions, and
clicked "accept". The models wrote the code, sourced the "data",
and composed the analysis.

If any information here is accurate, that is luck, not
determinism. Treat everything as somewhere between "plausible
starting point" and "confidently wrong". Verify before relying on
any claim, benchmark, or cost figure.

Built for no reason other than seeing what four chat models could
produce in a collaborative loop.
